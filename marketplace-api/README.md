# marketplace-api

This is a minimal scaffold for the Marketplace API (Express + Mongoose).

Quick start:

1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies: `npm install`.
3. Start server: `npm run dev` (requires `nodemon`) or `npm start`.

Files created: `config/`, `controllers/`, `models/`, `routes/`, `middleware/`, `services/`.
